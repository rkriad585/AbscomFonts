# Icons Reference

AbscomFonts includes 2492 icons across three font families: Brands, Solid, and Regular.

## Usage

```html
<i class="abs abs-{icon-name}"></i>
```

## Aliases

Many icons have aliases for convenience. For example:

- `abs abs-close` = `abs abs-xmark` = `abs abs-times`
- `abs abs-cog` = `abs abs-gear`
- `abs abs-home` = `abs abs-house`
- `abs abs-sync` = `abs abs-refresh`
- `abs abs-save` = `abs abs-floppy-disk`
- `abs abs-search` = `abs abs-magnifying-glass`

Both the short and full names work interchangeably.

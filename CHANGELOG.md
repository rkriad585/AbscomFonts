# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-06-29

### Added

- Initial release with 2492 icons across 4 font families (AbsBrands, AbsSolid, AbsRegular, AbsV4Compatibility)
- 30+ CSS animation classes with @keyframes definitions
- 22-color Tailwind-inspired palette with 11 shades each (550 color utility classes)
- Size classes (1x-16x) and relative sizing utilities
- Transform classes (rotate, flip)
- Layout utilities (flex, block, float, center)
- Decorative utilities (z-index, borders, hover effects)
- Built-in validation system with MutationObserver
- ESM, CJS, and IIFE builds
- Full TypeScript declarations
- Zero runtime dependencies
